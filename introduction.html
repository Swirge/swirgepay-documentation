Swirge Pay Intro

This site is a community run documentation site and is not managed by the official core team of Swirge Pay.

Swirge Pay is an automated market making (AMM) decentralized exchange (DEX) currently on the Ethereum blockchain. Unlike other protocols, Swirge Pay is a community run project that is governed by vote of the community. There are a few core products for Swirge Pay's ecosystem:

  • SwirgeSwap Exchange https://app.swirgepay.com
  • SwirgePay Yield Farming https://app.swirgepay.com
  • Swirge Pay Exchange https://app.swirgepay.com

Each of these serve a different purpose within the ecosystem.
