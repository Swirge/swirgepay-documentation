Swirge Pay Farms

The SushiSwap Farms (Menu) are located here:

  • #swirge farm url

The Farm allows users to yield farm Swirge rewards with each new block based on staking SLP tokens they received from app.swirgepay.com/pool
The farm previously supported UNI-V2 LP tokens from Uniswap but it now only works with tokens from SushiSwap pools.

Bootstrap source code

The Bootstrap source code download includes the precompiled CSS and JavaScript assets, along with source Sass, JavaScript, and documentation.

The scss/ and js/ are the source code for our CSS and JavaScript. The dist/ folder includes everything listed in the precompiled download section above. The site/docs/ folder includes the source code for our documentation, and examples/ of Bootstrap usage. Beyond that, any other included file provides support for packages, license information, and development.
